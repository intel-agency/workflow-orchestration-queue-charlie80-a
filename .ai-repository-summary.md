## Build Commands

### Install Dependencies

```bash
uv sync --dev
```

### Run Application

```bash
# Notifier (webhook receiver)
uv run os-apow-notifier

# Sentinel (orchestrator)
uv run os-apow-sentinel
```

### Run Tests

```bash
# All tests
uv run pytest

# With coverage
uv run pytest --cov=os_apow --cov-report=term-missing
```

### Code Quality

```bash
# Lint
uv run ruff check src/ tests/

# Format check
uv run ruff format --check src/ tests/

# Type check
uv run mypy src/
```

### Docker

```bash
# Build image
docker build -t os-apow .

# Run notifier
docker compose --profile notifier up

# Run all services
docker compose --profile all up
```

## Architecture (4-Pillar Model)

1. **Ear (Notifier)** - `src/os_apow/notifier/service.py`
   - FastAPI webhook receiver
   - HMAC signature validation
   - Event triaging

2. **State (Queue)** - `src/os_apow/queue/github_queue.py`
   - GitHub Issues as state store
   - Labels as status indicators
   - ITaskQueue interface for provider swapping

3. **Brain (Sentinel)** - `src/os_apow/orchestrator/sentinel.py`
   - Polling daemon (60s interval)
   - Assign-then-verify locking
   - Shell-bridge dispatch

4. **Hands (Worker)** - `scripts/devcontainer-opencode.sh`
   - DevContainer-based execution
   - opencode CLI integration

## Configuration

Required environment variables:
- `GITHUB_TOKEN` - GitHub PAT or App token
- `WEBHOOK_SECRET` - For notifier signature validation
- `GITHUB_ORG` - Target organization
- `GITHUB_REPO` - Target repository

Optional:
- `SENTINEL_BOT_LOGIN` - Bot account for locking
- `POLL_INTERVAL` - Polling interval (default: 60s)
- `HEARTBEAT_INTERVAL` - Heartbeat interval (default: 300s)
